üöÄ Quick Start

  1. Configure Environment Variables

    Update your .env file with email credentials:

    NODE_MAILER_USER=your_email@gmail.com NODE_MAILER_PASS=your_app_password NODE_MAILER_SERVICE=gmail NODE_MAILER_HOST=smtp.gmail.com NODE_MAILER_PORT=587
  2. Test Email Configuration

    Run the test script to verify setup:

    node test-email.js
  3. Start the Server

    Launch your backend server:

    npm start
  4. Access the OTP Form

    Open the password reset form in your browser:

    http://localhost:5000/otp-form.html

üìÆ API Endpoints

Request Password Reset:

POST /api/auth/request-password-reset Content-Type: application/json { "email": "user@example.com" }

Reset Password with OTP:

POST /api/auth/reset-password Content-Type: application/json { "email": "user@example.com", "otp": "123456", "newPassword": "newSecurePassword" }

üìß Gmail Users - Important!

To use Gmail, you need to generate an App Password:

  1. Go to Google Account Settings
  2. Navigate to Security ‚Üí 2-Step Verification
  3. Generate App Password for "Mail"
  4. Use this password in NODE_MAILER_PASS

üß™ Testing the System

  1. Create a Test User
